# Legacy Messages From ClearPass

This folder is retained as a legacy archive of one-way ClearPass-to-SDK handoff
notes.

Use [`../cross-repo-comms/`](../cross-repo-comms/) for new repo-to-repo or
agent-to-agent notes. Live notes must have sibling files in every corresponding
repo with the same `thread_id`, so git status can notify both sides when a
thread changes.
